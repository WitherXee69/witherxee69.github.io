#About Me

Hi, I’m Thisitha Sidartha aka WitherXee

I'm a passionate learner exploring the world of software development one project at a time. I love building tools, bots, and even programming languages — not just to code, but to understand how everything works.

My journey started with curiosity and turned into creation:

I built VISE, my first voice-based chatbot
I created LEO, a Python hacking toolkit
I’m developing Nyrox, my own Python based interpreted language
And I'm developing NEXA, A shell that can handle both Linux, Windows and its own unique commands

When I’m not coding, I’m writing stories, building worlds, or vibing to classic soft rock (70s–90s).

#Projects

NEXA_Shell

@Python @CLI @Shell

NEXA is a cross-platform shell environment designed to provide an easy and flexible command-line experience for developers and users.

LEO

@Python @Security

LEO (Lightweight Exploit Operator) is an all-in-one Python hacking toolkit designed for research, testing, and automation purposes.

Nyrox

@Python @Interpreted

Nyrox is an interpreted programming language written in Python, built from scratch as a personal learning project to understand how languages work behind the scenes — from lexers to parsers and runtime logic.

VISE

@Python @TTS @GPT

VISE (Voice Interactive Synthetic Entity) is a Python-based text-to-speech chatbot project built with the goal of exploring natural language interaction and synthetic voice responses.
